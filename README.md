# career-engine
Full-stack job application tracker with AI insights

## Built With
- **Frontend:** React, JSX, CSS  
- **Backend:** Python, Flask, RESTful APIs  
- **Database:** PostgreSQL  
- **Other:** OpenAI GPT-3.5 Turbo API, Werkzeug encryption  
- **Hosting:** Railway for Flask server and PostgreSQL database  

## Features
- Secure user authentication with hashed passwords
- End-to-end CRUD workflows for job applications
- Responsive React frontend with reusable components and protected routes
- AI-generated cover letter feedback using OpenAI's GPT-3.5 Turbo API
- Dashboard page for tracking progress and status of applications
